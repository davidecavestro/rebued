Concetti generali

URBE è un editor per risorse di localizzazione del software testuali scritto in Java.

Si tratta di uno strumento di supporto agli sviluppatori software; esso facilita la stesura delle risorse di localizzazione tipicamente chiamate nel gergo Java “file di properties” 1.

Per ulteriori informazioni sulle risorse di localizzazione, si veda il Java Tutorial.



Per iniziare

Se avete bisogno di questa applicazione, probabilmente non già avete na buona idea circa i concetti base del supporto alla localizzazione fornito dall'ambiente Java.

Ad ogni modo, ecco un breve glossario che può tornarvi utile per accedere a questo tutorial:

  • il bundle di localizzazione è un file di properties contenente le risorse relative ad un determinato Locale.

  • il Locale è un identificativo di Lingua/Regione /Variante a cui si possono associare delle traduzioni localizzate.



Risoluzione problemi

Per qualsiasi chiarimento o segnalazione inviare una mail all'indirizzo dell'autore.



Codice di terze parti

Questa applicazione usa librerie prodotte da terzi.

Fare riferimento alla directory <URBE-PATH>/license per ulteriori informazioni in merito.



1I file di properties sono facilmente riconoscibili dall'estensione “.properties”. Essisono anche usati come descrittori di configurazione per applicazioni, nel qual caso difficilmente hanno bisogno di essere localizzati.
